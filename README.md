
# # 📄 PDF Merge Web App

A simple, fast, and beginner-friendly **web application** that allows users to upload multiple PDF files and merge them into a single PDF — all in one click.


<h1 align="center">PDF Merge Web App</h1>
 
<p align="center">
  A lightweight web application to merge multiple PDF files into a single PDF.
</p>

Built using **Python**, **Flask**, and **PyPDF2**.

## ✨ Features
  
- 📂 Upload multiple PDF files at once  
- 🔗 Merge all PDFs into a single file  
- ⚡ Fast and lightweight  
- 🌐 Works on laptop and mobile browser  
- 🧑‍🎓 Perfect for students & beginners   
- 💻 Clean and simple user interface    

--- 

## 🛠️ Tech Stack       

- **Python**  
- **Flask** (Backend)
- **PyPDF2** (PDF processing)
- **HTML** (Frontend)
